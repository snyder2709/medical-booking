# МедБукинг
Медицинское веб-приложение для записи на консультации к врачам

## Технологии

- Nuxt 3
- TypeScript
- shadcn/ui
- Tailwind CSS
- Pinia
- VueUse

## Запуск и сборка

```bash
# Установка зависимостей
pnpm install

# Запуск dev сервера
pnpm dev

# Сборка приложения
pnpm build

# Предпросмотр production сборки
pnpm preview
```

## Заметки по рефакторингу и оптимизациям

1. Вынести инициализацию specialty в плагин preload
2. Composable для синхронизации url, кастомный useRouteQuery с оптимизацией под ssr
3. Сделать утилиту для кэширование запросов в useApi
4. Добавить автоимпорт для типов
5. Сделать скелетон компононет конструктор для pending
