# МедБукинг
Медицинское веб-приложение для записи на консультации к врачам

## Технологии

- Nuxt 3
- TypeScript
- shadcn/ui
- Tailwind CSS
- Pinia
- VueUse

## Запуск и сборка

```bash
# Установка зависимостей
pnpm install

# Запуск dev сервера
pnpm dev

# Сборка приложения
pnpm build

# Предпросмотр production сборки
pnpm preview
```
